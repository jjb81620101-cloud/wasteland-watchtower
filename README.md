# 廢土守夜站 v6

原創單檔手機網頁遊戲：基地放置補給、廣播接收角色、守夜班位編排，以及帶角色職能與手動落點技能的十波縱深塔防。

## 遊玩方式

1. 在「基地」領取離線補給並升級設施。
2. 在「頻道」接收新倖存者，在「倖存者」選三人小隊。
3. 調整近線哨、測距席、中繼席的順序，觀察小隊協定與角色特性。
4. 進入「防線」開始十波守夜；角色自動射擊，玩家處理最近威脅、技能落點與戰中資源。
5. 擊殺取得戰地零件，可召集守夜人、校準火力；第 3、6、9 波選擇一項強化。

GitHub Pages 入口為 `index.html`。遊戲無外部資產、外部依賴或追蹤碼，進度只保存在瀏覽器本機。
