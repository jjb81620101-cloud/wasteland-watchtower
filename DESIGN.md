# 廢土守夜站 v3 — 設計規則

## 視覺語言

- **煤灰** `#171713` 作為背景，不用紫黑漸層；**鐵鏽** `#a64b32` 是危險與攻擊；**舊紙** `#e7d7b0` 承載標籤；**警示黃** `#f2bd45` 只給可領取與主行動；**儀表綠** `#9abd6c` 表示安全、已完成與能量。
- 系統字體以「微軟正黑體 / Noto Sans TC fallback」為正文，等寬字體只用於頻率、數值、裝備編號。
- 以「控制台、打字標籤、鐵皮面板、訊號線」建立層級；面板圓角小、邊框有用途，不做 dashboard 卡片海。

## 元件與動態

- 首頁只有一個主視覺：廣播塔訊號掃描。雷達環、掃描線和頻率條共同工作，不能為裝飾而散落動畫。
- 底部導覽固定為：基地、遠征、倖存者、頻道。每頁一個主要行動，獎勵與升級一律有立即回饋。
- 稀有度：灰=C、藍=R、黃=SR、鐵鏽紅=SSR。數值與資源使用一致色彩。
- 所有動態須在 350ms 內；戰鬥保留一次約 2.4 秒的結果節奏。`prefers-reduced-motion` 下停止掃描與轉場。

## 禁止事項

- 不使用外部圖片、字型、CDN、框架、追蹤碼或既有遊戲素材。
- 不做網頁儀表板格線、紫藍霓虹、巨大漸層 KPI、無意義玻璃卡。
- 不讓按鈕沒有狀態：資源不足須禁用，執行中須鎖定，結果要以彈層或提示回應。
