# 廢土守夜站 v7｜戰地圖鑑與夜間訊號站

## 產品核心

v6 延續「基地放置補給 → 廣播接收角色 → 編排守夜班位 → 十波縱深守線 → 取得資源再養成」的單手手機循環。這一版的主題不是增加更多角色卡，而是讓角色職能、個人特性與班位真正改變戰鬥。

v7 是使用者明確要求的美術升級：角色改為可見的臉，戰場改為可見的攻擊。玩法規則與 v6 完全相同，只新增圖鑑肖像與戰鬥動畫兩層表現。

## 角色玩法

- 每名角色有原創呼號、火力／控場／後勤職能、個人特性、主動技能、等級與星級。
- 三個起始班位為近線哨、測距席與中繼席，分別強化自動普攻、技能半徑、冷卻／修復。
- 玩家可在角色頁調整班位順序；重新排班會重置尚未完成的當局，避免中途換位套利。
- 職能組合啟動一項協定：三職能提高全傷害與指令、雙火力提高普攻、雙控場提高範圍與減速、雙後勤提高修復與回收。
- 個人特性只改動可解釋的戰鬥變數，例如近線傷害、重裝傷害、技能半徑、修復、回收或召集成本。

## 戰鬥資訊層級

1. 最近威脅與預估接觸時間。
2. 敵群位置、種類與生命。
3. 塔芯耐久、波數、戰地零件與指令。
4. 七格火線的角色、班位效果、技能冷卻。
5. 召集、校準、修復與波間三選一。

## 美術介面

- 主題是「夜間訊號站的戰地檔案」，不是來源影片的明亮奇幻角色卡。
- 主色：夜藍綠 `#091011`、檔案紙 `#e8dfc8`、警示黃 `#efb34b`、鐵鏽紅 `#c25d42`、儀表綠 `#8fc58d`、測距藍 `#75aab7`。
- **v7（使用者明確要求覆蓋 v6 的「不使用外部角色圖」原則）**：12 名角色對應 9 組 AI 產生的原創肖像（Pollinations 生成、ink/wash 風格、同一палette，256px 壓縮為 JPEG 並以 base64 data URI 內嵌，單張 ≤25KB），顯示於角色頁卡片、班位卡與召喚結果；肖像缺漏時自動退回原本的呼號字牌。七格火線的守夜人改為程式繪製的 SVG 小人（依職能色染色，頭部裁切嵌入肖像臉孔），具待機浮動；每次自動攻擊有後座力動畫、彈道曳光、命中閃光與火花，技能施放有職能色擴散光環，敵人陣亡有溶解動畫。CSS 地形剪影與測距燈仍是背景基底視覺。
- 所有新增動畫皆為純 CSS/SVG/Canvas 程式繪製，不使用任何外部圖片或影片素材；肖像以外沒有任何網路圖庫依賴。
- 頂部三資源、中央任務、底部四分頁；彈窗只用於獎勵、確認與波間選擇。
- 按鈕、禁用、冷卻與危險皆同時有文字；不只靠顏色。

## 技術與邊界

- 唯一執行檔為 `index.html`，內含 HTML、CSS、JavaScript。
- 無 npm、框架、CDN、外部字型、追蹤碼與執行期網路請求；肖像等圖片一律以 data URI 內嵌，不對外部主機發出請求（v7 沿用此鐵律，僅新增內嵌圖片，未新增任何 fetch/XHR/script src）。
- 存檔鍵為 `wasteland-watchtower-v7`，可讀取 v4–v6 舊存檔並遷移；戰中暫態不寫入存檔。
- 全部互動使用原生 `button`，支援鍵盤焦點；`prefers-reduced-motion` 會停用動畫（含 v7 新增的戰鬥特效層，該模式下直接不產生特效節點，回到純狀態變化）。
- 目標寬度 320–430px，桌機只放大外框，不改變單手操作順序。

## v6 驗收條件

- 角色頁能看到並操作三個班位，調序後順序立即改變。
- 戰鬥開始後，角色按鈕包含班位與技能名稱，威脅 ETA 隨敵人推進更新。
- 手動技能可選角色再點落點；戰鬥紀錄顯示角色、班位、技能與結果。
- 第 3、6、9 波出現三選一，選擇後自動續戰。
- 手機與桌機載入無 console error / warning；頁面不發出外部請求。

## v7 驗收條件

- 角色頁、班位卡與召喚結果顯示肖像圖片，缺漏時退回呼號字牌，不出現破圖。
- 戰鬥中七格火線顯示程式繪製小人，依職能色染色，待機有浮動。
- 自動攻擊時可觀察到曳光、後座力與命中特效；技能施放可觀察到職能色光環；敵人陣亡有消散效果。
- `prefers-reduced-motion` 開啟時特效不產生，僅保留狀態文字與數值更新。
- 頁面仍只發出一次對本機的 HTML 請求，無任何外部網路請求；console 無 error / warning。
